# AutoMec — Site Institucional

> Desenvolvido por **Autotech Sistemas**

---

## O que é esse projeto?

Site modelo desenvolvido pela **Autotech Sistemas** para demonstrar como uma oficina mecânica pode se apresentar digitalmente. Todos os dados exibidos — nome, endereços, telefones e unidades — são fictícios e servem apenas como exemplo. Para um cliente real, o conteúdo é adaptado com as informações verdadeiras do negócio.

---

## O que o site oferece ao cliente final?

- Apresentação clara dos serviços oferecidos pela oficina
- Galeria de marcas atendidas (BMW, Audi, Mercedes, Porsche, Toyota, Volvo, Jeep)
- Seção de depoimentos de clientes reais
- Perguntas frequentes para tirar dúvidas antes de ligar
- Localização das três unidades com mapa integrado do Google
- Formulário de agendamento e botão de contato direto pelo WhatsApp

## Diferenciais do projeto

**Aparece no Google.** O site foi construído com boas práticas de SEO local — isso significa que ele tem mais chances de aparecer quando alguém pesquisar "oficina mecânica em Uberlândia" ou cidades próximas.

**Funciona em qualquer dispositivo.** O layout se adapta automaticamente a celulares, tablets e computadores.

**Visual profissional.** Design moderno com animações, galeria de fotos e identidade visual consistente — passa credibilidade para quem visita.

## O que precisa ser fornecido pelo cliente para finalizar

Antes de publicar, o cliente deve entregar:

- [ ] Logotipo em PNG com fundo transparente
- [ ] Fotos das fachadas das três unidades
- [ ] Fotos dos carros em PNG sem fundo (para as ilustrações do site)
- [ ] Endereços completos de cada unidade
- [ ] Telefones reais de cada unidade
- [ ] Confirmação do e-mail de contato
- [ ] Domínio onde o site será publicado
